robotframework-uiautomatorlibrary
=============

This is a test library for [Robot Framework](https://pypi.python.org/pypi/robotframework).

It extend the test ability for Android Device by using [Python uiautomator](https://pypi.python.org/pypi/uiautomator) internally.

# Installation #

	$ pip install robotframework-uiautomatorlibrary

# Getting started #
